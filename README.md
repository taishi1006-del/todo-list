# TODO List

VS Code でそのまま開いて使える、シンプルな TODO アプリです。

## 公開URL

GitHub Pages が有効になったら、ここから開けます。

- [https://taishi1006-del.github.io/todo-list/](https://taishi1006-del.github.io/todo-list/)

## 使い方

1. `index.html` をブラウザで開く
2. タスクを入力して `追加`
3. 完了チェック、削除、フィルタ表示を利用する

## 機能

- タスク追加
- 完了切り替え
- 個別削除
- 完了済みの一括削除
- `localStorage` による自動保存
